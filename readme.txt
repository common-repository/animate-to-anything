Animate to Anything using class or id

== Description ==
You can animate any element of your website by using its class or id. a large no. of animations and easy to use. Make your website more attractive and playfull.

== Installation ==

1. Add Plugin to your wordpress
2. Go to Admin Menu->Animation
3. Fill Animation form
4. Save form .
5. Now it will appear on wordpress .
